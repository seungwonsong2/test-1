## 명령 연습
- pwd : 현재 위치 경로
- ls : 현재 디렉토리 내 파일,경로
  - ls -a : 숨긴 파일,디렉토리 표시
  - ls -l : 파일이나 디렉토리 상세정보 표시
  - ls -r : 정렬 순서 거꾸로
  - ls -t : 파일 작성 시간순 나열
- clear : 터미널 창 지우기(라샛)
- cd .. : 상위 디렉토리 이동
- cd ~ :  홈 디렉토리 이동
- mkdir (이름) : 디렉토리 생성
- rm -r (이름) : 디렉토리 삭제
- exit :  터미널 종료
